# AI Career & Freelancing Assistant

## Project Overview

This project helps students explore IT careers, track skills, analyze freelancing earnings, and receive AI-based career recommendations.

## Features

* Excel Dashboard
* Career Analysis
* Freelancing Earnings Tracking
* Charts and Pivot Tables
* VLOOKUP/XLOOKUP
* Python Career Recommendation Program
* AI Career Suggestions

## Tools Used

* Microsoft Excel
* Python
* ChatGPT
* GitHub
